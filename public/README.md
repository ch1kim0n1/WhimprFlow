# WhimprFlow Brand Assets

The logo mark represents a spoken signal resolving into a continuous flow. It uses the product's existing deep-slate and aqua palette.

## Primary Assets

- `whimprflow-mark-transparent.png`: transparent master mark for light or dark layouts.
- `whimprflow-mark-512.png` and `whimprflow-mark-256.png`: transparent raster exports.
- `whimprflow-app-icon-1024.png`: opaque master app icon with the mark on a deep-slate surface.
- `whimprflow-app-icon-512.png` and `whimprflow-app-icon-256.png`: app-icon exports.
- `whimprflow-logo-dark.svg`: horizontal lockup for light backgrounds.
- `whimprflow-logo-light.svg`: horizontal lockup for dark backgrounds.
- `whimprflow-logo-dark.png`: transparent horizontal lockup for light backgrounds.
- `whimprflow-logo-light.png`: transparent horizontal lockup for dark backgrounds.

## Palette

- Deep slate: `#111419`
- Aqua: `#22C3B6`
- Bright aqua: `#3FE0D0`
- Light text: `#F7F9FB`

The transparent mark is the canonical logo. Do not recolor its aqua ribbon or flatten it onto a colored background without preserving clear contrast.
